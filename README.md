# TODOアプリ

React + TypeScript + Vite + Tailwind CSS で作られたシンプルなTODOアプリです。データはブラウザの localStorage に保存されます。

## 機能

- タスクの追加・削除
- 完了/未完了の切り替え
- タスクのダブルクリックによるインライン編集
- フィルタ表示(すべて / 未完了 / 完了)
- 完了済みタスクの一括削除
- localStorage によるデータ永続化

## 使い方

```bash
npm install
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
npm run lint     # Lint実行
```
